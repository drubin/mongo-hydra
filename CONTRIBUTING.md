# Contributing Guide

## CLI

In order to make development and testing the CLI tool you can run `npm link` in the project root.

This will allow you to execute the CLI commands as if they were globally installed.

```sh
$ hydra -h
hydra <command>
....
Hail Hydra
copyright 2020 unu GmbH

$ mongo-hydra -h
hydra <command>
....
Hail Hydra
copyright 2020 unu GmbH
```
